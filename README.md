# GrabDocs Testing Assignment

## Course
COSC369-400 | Software Engineering | Spring 2026

## Overview
This repository contains automated tests for the GrabDocs platform
using Cypress. Three features are tested: Feedback, Workspace, and Forms.

## Features Tested
- Feedback (TC-FB-01, TC-FB-02, TC-FB-03)
- Workspace (TC-WS-01, TC-WS-02, TC-WS-03, TC-WS-04)
- Forms (TC-FM-01, TC-FM-02, TC-FM-03, TC-FM-04)

## Tools Used
- Cypress (Automated Testing)
- JavaScript
- GrabDocs (https://app.grabdocs.com)
- JIRA (Requirements Management)

## How to Run Tests
1. Clone this repository
2. Run `npm install`
3. Run `npx cypress open`
4. Select `grabdocs.cy.js` and run
